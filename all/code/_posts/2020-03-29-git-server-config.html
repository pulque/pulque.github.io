在Win10上搭建并配置Git服务器，在局域网内进行代码提交。
也可以保存文件，方便、快捷、高效。

1.服务器工具：gitblit

官网： http://www.gitblit.com/
无需安装，解压即可。

2.修改配置文件

目录：gitblit-1.9.0\data\defaults.properties

2.1：服务器仓库本地目录：

git.repositoriesFolder = E:/Git/GitRepository

2.2：服务器访问端口：

server.httpPort = 9874

2.3：服务器IP地址：

server.httpBindInterface = 192.168.1.100
保存文件

3.开启服务

执行：\gitblit-1.9.0\gitblit.cmd

4.网页端配置Git

浏览器打开：http://192.168.1.100:9874/
可以新建或修改仓库。
