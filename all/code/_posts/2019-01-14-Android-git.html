为了能够随时随地向git仓库中提交文件，希望在手机上能做到这点。
就找到了开源termux终端APP，类似“超级终端”APP，
和在Linux上使用终端一样方便。
在termux的基础上修改，添加辅助输入，编辑文件。

git安装

1.apt update && apt upgrade

更新源和升级，否则安装git会失败。

2.apt install git

安装git。

3.termux-setup-storage

映射目录，这样就可以操作SDCard上的文件了。
加上具体的目录映射：（先要建立相应的文件夹在SDCard上）
ln -s /data/data/com.termux/files/home/storage/shared/github

添加功能

1.一些常用命令

如“cd ..”、“git clone”、“git pull”、等等
点击按钮就可以代替输入。

2.在手机上编辑文件

其实可以使用手机上的其他app编辑，后续可能会添加其他实用功能，根据需求修改。

3.记录已经输入过的文字

记录输入过的文字和标签，可以帮助用户快速输入或修改。
