7 Apr 2020

Android Studio使用中，Build框中出现乱码。

解决：

C:\Users\【用户名】\.AndroidStudio3.6\config目路下，
studio64.exe.vmoptions文件，用文本打开，
并添加，“-Dfile.encoding=UTF-8”，保存重启即可。
如果没有此目路，在安装目录下查找即可。
